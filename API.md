## swap 区使用情况
### 地址： 
> ip:port/swap/info

### 请求方法：
GET

### 返回参数：
```javascript
{
  "total": 1073741824,
  "used": 4456448,
  "free": 1069285376,
  "usedPercent": 0.4150390625,
  "sin": 0,
  "sout": 0
}
```
